Tecnologias com que trabalho:

Oracle APEX
O APEX é minha principal stack. É um framework extremamente poderoso onde se produz muito mais, com menos.

Comecei minha tragetória no APEX 5.1 e pude vislumbrar a grande evolução da ferramenta ao longo do tempo. Hoje desenvolvo nas versões mais atuais 🚀

PL/SQL
O PL/SQL é a minha especialidade no backend.

Possuo experiência desenvolvendo grandes projetos utilizando esta poderosa linguagem para automatizar operações e constuir processos 💪🏽

ORDS / API
Com o APEX podemos construir API's RESTful em pouquíssimo tempo, viabilizando a comunicação seguindo as boas práticas atuais.

Tenho experiência em construção de API's nos métodos GET, POST e PUT além de constuir formas de autenticação utilizando o protocolo Oauth 2.0. 🗣

Javascript
O JS é uma das stacks que eu mais me interesso devido ao seu uso variado e sua adesão ao longo do tempo.

Além das experiências que tive utilizando JS nas empresas que trabalhei, também possuo diversos projetos pessoais utilizando-o em contextos diferentes 🚀

HTML
Se boa parte da web é construída com JS, toda web é construída com HTML.

Por isso, sempre busco trabalhar utilizando as melhores práticas desta tecnologia 💡

CSS
Assim como o HTML constrói, o CSS estiliza a web.

Busco usar o CSS com criatividade, respeitando as boas práticas, melhores modelos de layout e acessibilidade ♿

Empresas que já trabalhei:

CR Diementz

A equipe era formada por 3 integrantes, na qual eu era o principal incubido do desenvolvimento do software "MK Portal". O Software possui centenas de usuários e atende toda rede interna da MK Química. Foi desenvolvido em APEX 20.2, versão na qual mais me aprofundei.

Mout's TI

Trabalhei em uma squad com outros 2 Devs desenvolvendo o software "Portal do Fornecedor".
O Portal do Fornecedor é um software transacional construído em APEX que conta com diversas funcionalidades interativas e milhares de usuários que são, em sua maioria, de fora da rede interna da empresa.
Em termos de devOps, elaborei aceites técnicos, pratiquei code review, realizei subidas para o ambiente de produção (releases) e outras ações de liderança no projeto.
Também colaborei com o desenvolvimento das primeiras API's do Portal do Fornecedor, que contam com autenticação Oauth 2.0 e seguem todos os padrões REST exigidos hoje no mercado.
Segui a metodologia ágil SCRUM, utilizando a plataforma Microsoft Azure para o controle.

Tecnologias Utilizadas: Oracle APEX, PL/SQL, SQL, HTML 5, CSS 3, JavaScript & bibliotecas e Git/GitHub.
